# Anki Telegram Bot

Telegram-бот для создания карточек Anki с помощью Gemini AI.  
Отправляешь слово → получаешь готовую карточку → сохраняешь в Anki одной кнопкой.

Поддерживаемые языки: **японский, русский, английский**.

---

## Как это работает

1. Отправляешь боту слово или фразу
2. Бот определяет язык источника
   - В режиме **Многоязычный** — предлагает выбрать язык перевода кнопками 🇯🇵 🇷🇺 🇬🇧
   - В режиме **RU ↔ JP** или **RU ↔ EN** — целевой язык определяется автоматически
3. Gemini генерирует карточку: слово, чтение, перевод, пример с переводом
4. Нажимаешь **✓ Добавить** — карточка уходит в Anki и сохраняется в файл  
   Нажимаешь **✕ Пропустить** — карточка отклоняется

**Команды:**

| Команда | Описание |
|---|---|
| `/start` | Приветствие |
| `/card` | Выбрать формат карточки: **Изучаемый → Основной** или **Основной → Изучаемый** (по умолчанию). Выбор сохраняется между перезапусками |
| `/mode` | Выбрать языковой режим: **🌍 Многоязычный** (по умолчанию), **RU ↔ JP**, **RU ↔ EN**. Выбор сохраняется между перезапусками |
| `/help` | Краткая справка |

---

## Требования

- **Java 21**
- **Anki** (десктопное приложение) — должен быть открыт во время работы бота
- **AnkiConnect** — плагин для Anki
- Аккаунт в **Telegram**
- Аккаунт в **Google AI Studio** для Gemini API

---

## Шаг 1 — Создать Telegram-бота

1. Открой Telegram, найди [@BotFather](https://t.me/BotFather)
2. Отправь `/newbot`
3. Придумай название бота (например: `My Anki Bot`)
4. Придумай username — должен заканчиваться на `bot` (например: `my_anki_bot`)
5. BotFather пришлёт **токен** вида `123456789:AAF...` — сохрани его, это `BOT_TOKEN`
6. Username бота (без `@`) — это `BOT_USERNAME`

---

## Шаг 2 — Узнать свой Telegram Chat ID

1. Найди в Telegram [@userinfobot](https://t.me/userinfobot)
2. Нажми `/start`
3. Бот пришлёт твой `Id` — числовое значение, это `ALLOWED_CHAT_ID`

---

## Шаг 3 — Получить Gemini API ключ

1. Перейди на [aistudio.google.com](https://aistudio.google.com)
2. Войди с Google-аккаунтом
3. В левом меню нажми **Get API key**
4. Нажми **Create API key**
5. Скопируй ключ — это `GEMINI_API_KEY`

---

## Шаг 4 — Установить AnkiConnect

1. Открой Anki
2. Меню **Tools → Add-ons → Get Add-ons**
3. Введи код: `2055492159`
4. Перезапусти Anki
5. Anki должен быть открыт всё время, пока работает бот

---

## Шаг 5 — Подготовить колоды и тип карточек в Anki

1. Создай две колоды — одну для японского, одну для английского  
   (названия запомни, они понадобятся ниже)
2. Убедись, что в Anki есть тип карточки с полями **Front** и **Back**  
   Запомни точное название типа — это `ANKI_MODEL_NAME`

> Тип карточки по умолчанию называется **Простая** (если Anki на русском)  
> или **Basic** (если на английском)

---

## Шаг 6 — Настроить переменные окружения

В IntelliJ IDEA: **Run → Edit Configurations → Environment variables**

| Переменная | Значение | Где взять |
|---|---|---|
| `BOT_TOKEN` | `123456789:AAF...` | BotFather (шаг 1) |
| `BOT_USERNAME` | `my_anki_bot` | BotFather (шаг 1) |
| `ALLOWED_CHAT_ID` | `123456789` | @userinfobot (шаг 2) |
| `GEMINI_API_KEY` | `AIza...` | Google AI Studio (шаг 3) |
| `DECK_NAME_JP` | `Japanese` | Название колоды для японского |
| `DECK_NAME_ENG` | `English` | Название колоды для английского |
| `ANKI_MODEL_NAME` | `Простая` | Тип карточки в Anki (шаг 5) |

Необязательные (есть значения по умолчанию):

| Переменная | По умолчанию | Описание |
|---|---|---|
| `GEMINI_MODEL` | `gemini-flash-lite-latest` | Модель Gemini |
| `GEMINI_URL` | `https://generativelanguage.googleapis.com/v1beta/models/` | URL API |

---

## Шаг 7 — Запустить

1. Убедись, что **Anki открыт**
2. Запусти приложение через IntelliJ: **Run → Run 'AnkiTelegramBotApplication'**
3. В логах должна появиться строка:
   ```
   Бот зарегистрирован: @your_bot_username
   ```
4. Открой бота в Telegram и отправь `/start`

---

## Файл с карточками

Все добавленные карточки дублируются в файл `anki_cards.txt` в корне проекта.  
Формат: `Front[TAB]Back` — в том же формате, что выбран командой `/card`.  
Файл можно импортировать в Anki вручную через **File → Import**.

## Возможные проблемы

**❌ Бот не отвечает**  
→ Проверь, что все переменные окружения заданы и приложение перезапущено

**❌ Лимит запросов к Gemini исчерпан. Попробуй позже.**  
→ Исчерпан лимит бесплатных запросов. Подожди или смени API ключ

**❌ Тип карточки не найден. Проверь ANKI_MODEL_NAME**  
→ Название типа карточки в `ANKI_MODEL_NAME` не совпадает с тем, что в Anki

**❌ Колода не найдена. Проверь DECK_NAME_JP / DECK_NAME_ENG**  
→ Название колоды не совпадает с тем, что в Anki

**❌ Не удалось сохранить карточку в Anki**  
→ Anki не открыт или плагин AnkiConnect не установлен

**❌ Бот запустился, но не регистрируется**  
→ Проверь `BOT_TOKEN` — он должен быть полным, включая цифры до двоеточия
